# Kottenpark_Minecraft_Mod
Minecraft made of our school, Kottenpark, for an assignment

world playable at: minecraft.koolwaaij.nl:25566

to play: install the jar file from https://github.com/Opal-Skies/Kottenpark_Minecraft_Mod/tree/main/build/libs
play using minecraft fabric and the dependencies
Dependencies:
* Geckolib      https://modrinth.com/mod/geckolib
* Fabric API    https://modrinth.com/mod/fabric-api
